# Usage
<!-- usage -->
```sh-session
$ npm install -g ism-cli
$ ism COMMAND
running command...
$ ism (-v|--version|version)
ism-cli/0.0.0 win32-x64 node-v12.14.1
$ ism --help [COMMAND]
USAGE
  $ ism COMMAND
...
```
